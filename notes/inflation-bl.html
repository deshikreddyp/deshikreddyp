Inflation BL

In setting up cases for Reynolds averaged Navier-Stokes (RANS) analysis for CFD, we usually incorporate a prism layer along wall surfaces to capture boundary later effects in better detail. The technique in such cases would be to size the first layer of mesh for a target y+ value

As the region closer to wall have significant velocity gradients, we need to model it in detail. So we have to discretize the near surface region more cautiously and finer. This can be done by inflating the cells near wall boundary. As you might have observed in Ansys or Foam, the software asks for few values such as layer thickness, growth ratio etc.

Velocity Potential

In fluid dynamics, the Velocity Potential ($\phi$) is a scalar function used to describe irrotational flow. When a flow is irrotational, the velocity field $\mathbf{u}$ can be expressed as the gradient of this potential function.

Mathematical Definition

For an irrotational flow, the vorticity $\omega = \nabla \times \mathbf{u} = 0$. This condition allows us to define $\phi$ such that:

$$\mathbf{u} = \nabla \phi$$

In Cartesian coordinates $(x, y, z)$, the velocity components are:

$$u = \frac{\partial \phi}{\partial x}, \quad v = \frac{\partial \phi}{\partial y}, \quad w = \frac{\partial \phi}{\partial z}$$

Laplace's Equation

For an incompressible and irrotational flow, the continuity equation ($\nabla \cdot \mathbf{u} = 0$) leads to Laplace's equation for the velocity potential:

$$\nabla \cdot (\nabla \phi) = \nabla^2 \phi = 0$$

This is a fundamental result in potential flow theory, allowing complex flow fields to be solved using techniques from linear partial differential equations.

Relation to Bernoulli's Equation

For steady, irrotational, incompressible flow, the Bernoulli equation can be written in terms of the velocity potential:

$$\frac{\partial \phi}{\partial t} + \frac{1}{2}|\nabla \phi|^2 + \frac{p}{\rho} + gz = f(t)$$

💡 Note: While velocity potential is defined for irrotational flows in any number of dimensions, the stream function is typically only defined for 2D or axisymmetric flows.

The Generalized Minimal Residual (GMRES) Method

Category: Numerical Linear Algebra · Krylov Subspace Methods · GMRES · PETSc Solvers

GMRES (Generalized Minimal Residual) is an iterative method for solving a non-symmetric linear system $Ax = b$. It belongs to the class of Krylov subspace methods and is a standard choice for large, sparse systems where direct methods are computationally prohibitive.

Core Concept

Starting with an initial guess $x_0$, GMRES minimizes the residual norm $\|r_k\| = \|b - Ax_k\|$ over the $k$-th Krylov subspace:

\[ \mathcal{K}_k(A, r_0) = \text{span} \{ r_0, Ar_0, A^2 r_0, \dots, A^{k-1} r_0 \} \]

The method constructs an orthonormal basis for this subspace using the Arnoldi iteration, transforming the large system into a much smaller least-squares problem involving a Hessenberg matrix.

The GMRES Process

  1. Initialize: Compute $r_0 = b - Ax_0$.
  2. Arnoldi Iteration: Successively build the orthonormal basis vectors $v_1, v_2, \dots$ and the Hessenberg matrix $H_k$.
  3. Solve Minimization: Find $y$ that minimizes $\| \beta e_1 - \bar{H}_k y \|_2$ where $\beta = \|r_0\|$.
  4. Update: $x_k = x_0 + V_k y$.
Restarting (GMRES(m)).

Because the cost of orthogonalizing new vectors against all previous ones increases quadratically with $k$, GMRES is often "restarted" after $m$ steps. This prevents memory overflow but may slow down convergence or lead to stagnation.

Preconditioning

GMRES is rarely used without a preconditioner $M$. Instead of $Ax=b$, we solve $M^{-1} Ax = M^{-1} b$ (left preconditioning) or $AM^{-1} y = b$ (right preconditioning). A good preconditioner clusters the eigenvalues of the operator, leading to rapid convergence.

Implementation.

In PETSc, GMRES is the default solver for KSP (Krylov Subspace Package). Common configurations include -ksp_type gmres -pc_type gamg for monolithic FSI problems.

PETSc FieldSplit Preconditioner

Category: Linear Algebra/Solvers

Last updated: ...

These notes summarize the solver strategy for restarted FSI simulations using PETSc FieldSplit. In multiphysics problems, FieldSplit allows for block-based preconditioning by dividing the global Jacobian into logical sub-blocks (e.g., velocity and pressure).

Core Concept.

A global Jacobian $J$ is viewed as a block matrix:

$$J = \begin{bmatrix} A & B \\ C & D \end{bmatrix}$$

where $A$ is the velocity block, $D$ the pressure block, and $B, C$ are the coupling terms. FieldSplit leverages these blocks to apply specialized preconditioners (like Schur complements).

Block Factorization Variants

FieldSplit supports several factorization patterns for the Schur split:

  • Lower: Solve the velocity side first, then the Schur side.
  • Upper: Apply the coupling in the opposite triangular ordering. Often performs better if the coupling is physics-dominated in one direction.
  • Full: Strongest but most expensive; applies the full block factorization.

Observed Performance Ranking

In high-resolution FSI tests on 256 MPI ranks, various FieldSplit configurations were compared. The working ranking from the logs was:

fieldsplit4 (upper) > fieldsplit5 (full) > fieldsplit2 (lower) > fieldsplit6 >> fieldsplit3
Numerical Warning.

ALE Jacobians reporting strongly negative values (e.g., -6 to -8) indicate an inverted mesh. While solver comparisons remain informative, absolute timings are affected by the numerically unhealthy state of the mesh distortion.

Scaling Diagnostics

To understand if the solid coefficients dominated the Jacobian, block norms were analyzed across different solid shear modulus scales:

Scale Factor $\|J^{solid}_{VV}\|_F$ $\|J_{PP}\|_F$
$0.1$ $6.36 \times 10^1$ $6.47 \times 10^5$
$1.0$ $6.37 \times 10^2$ $6.47 \times 10^5$
$10.0$ $6.38 \times 10^3$ $6.47 \times 10^5$
Key takeaway.

The velocity block ($VV$) is solid-dominated, but the full Jacobian is globally dominated by the pressure block ($PP$). FieldSplit isolates the stiff solid-dominated velocity block, which can be both a strength and a bottleneck.

Conclusions

  1. The fieldsplit4 (Upper) variant was the most robust Schur configuration.
  2. The system's stiffness combines a stiff velocity block with a massive pressure block.
  3. Monolithic gamg often beats FieldSplit here because it treats the whole operator more holistically.

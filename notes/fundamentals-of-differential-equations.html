Fundamentals of Differential Equations

ODE: Ordinary Differential Equations

A differential equation whose unknown function is only dependent on ‘1’ variable.

PDE: Partial Differential Equations

Unknown function is dependent on 2 or more variables.

Order of DE (O/P) is highest degree derivative in it.

Homogeneous, IF u = 0, is solution of O/P DE.

Inhomogeneous, ELSE

A boundary condition is homogeneous if u = 0, satisfies it. ELSE it is inhomogeneous.

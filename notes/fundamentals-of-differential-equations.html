Fundamentals of Differential Equations

Category: Calculus · Differential Equations · Engineering Mathematics

Last updated: ...

ODE: Ordinary Differential Equations

A differential equation whose unknown function depends on only one independent variable.

PDE: Partial Differential Equations

A differential equation where the unknown function depends on two or more independent variables.

  • Order: The highest degree derivative present in the equation.
  • Homogeneous: If $u = 0$ is a solution to the equation.
  • Homogeneous Boundary Condition: If $u = 0$ satisfies the condition at the boundary.

Linear vs. Non-Linear DEs

In a linear equation, the dependent variable and its derivatives appear only to the first power and are not multiplied together. For example:

  • $x$ is linear, while $x^2$ is not.
  • Transcendental functions like $\sin(x)$ make an equation non-linear in $x$.

Examples of Linear Equations

\[ x'' + x = 0 \] \[ x'' + 2x' + x = 0 \] \[ x' + y' = 0 \]

Examples of Non-Linear Equations

\[ x' + 1/x = 0 \] \[ x' + x^2 = 0 \] \[ x'' + \sin(x) = 0 \] \[ x x' = 0 \]
Linearity with respect to variables.

Note that an equation can be linear in one variable but non-linear in another. For example, $\sin(t)x' + \cos(t)x = e^t$ is linear in $x$ but the coefficients are non-linear transformations of the independent variable $t$.

Theorems of Interest.

Existence and Uniqueness theorems, such as the Picard-Lindelöf theorem, provide conditions under which a first-order ODE has a unique solution.

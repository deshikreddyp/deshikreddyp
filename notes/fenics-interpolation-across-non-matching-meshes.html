FEniCS: Interpolation across non-matching meshes

This code demonstrates how to interpolate solution fields (specifically displacement fields) from one mesh to another in FEniCS. This process is often called "regridding" and is useful when:

  • You need to transfer simulation results to a different discretization
  • You want to continue a simulation on a refined or modified mesh
  • You need to compare or combine results from different simulations

Key Components

  1. Mesh Loading
    • Source mesh: The original mesh where displacement data exists
    • Target mesh: The new mesh where data will be interpolated to
    • Domain information is preserved during transfer
  1. Function Spaces
    • Vector function spaces are created on both meshes (CG1 elements)
    • set_allow_extrapolation(True) enables interpolation for points slightly outside the source domain
  1. Interpolation Process
    • For each time snapshot:
      1. Read displacement field from source mesh
      1. Interpolate to target mesh using LagrangeInterpolator
      1. Write interpolated field to output files
  1. Output Generation
    • HDF5 file containing all interpolated displacement fields
    • XDMF time series for easy visualization in ParaView

This interpolation technique preserves the mathematical properties of the original solution while transferring it to a new discretization, making it valuable for multi-stage simulations or post-processing workflows.

# regrid_displacement_only_no_defs.py
from dolfin import *
import os

# ────────────────────── MPI & logging ──────────────────────
COMM = MPI.comm_world
RANK = COMM.Get_rank()
from dolfin.cpp.log import set_log_active
set_log_active(RANK == 0)

# ────────────────────── EDIT THESE PATHS ───────────────────
# Mesh that was used for the original snapshots (checkpoint mesh)
SRC_MESH_H5   = "/home/dputluru/fenics_run/navier-stokes/clinical_trials/clinical_wo_injection.h5"

# NEW mesh to regrid onto (must contain /mesh and /domains)
NEW_MESH_H5   = "/depot/hgomezdi/data/dputluru/clinical_trials/pre-process/meshes/aswanth.h5"  # ← TODO

# Single HDF5 with periodic displacement snapshots under /displacement/{i}
SNAPSHOTS_H5  = "/scratch/negishi/dputluru/clinical_trials/C4_phase2_new/final_snapshots3.h5"

# Periodic timing info
PERIOD = 0.8
NUM_SNAPSHOTS = 64
DT_SNAP = PERIOD / float(NUM_SNAPSHOTS)

# Outputs
OUT_DIR       = "/scratch/negishi/dputluru/regridded_to_newmesh_phase2"
os.makedirs(OUT_DIR, exist_ok=True)
OUT_H5        = os.path.join(OUT_DIR, "regridded_displacement.h5")
OUT_XDMF_DISP = os.path.join(OUT_DIR, "displacement_timeseries.xdmf")

# ────────────────────── SOURCE (checkpoint) MESH ───────────
src_mesh = Mesh()
hdf_src = HDF5File(COMM, SRC_MESH_H5, "r")
hdf_src.read(src_mesh, "/mesh", False)
hdf_src.close()

V_src = VectorFunctionSpace(src_mesh, "CG", 1)  # displacement space on source mesh

# ────────────────────── TARGET (new) MESH + DOMAINS ────────
tgt_mesh = Mesh()
hdf_tgt_in = HDF5File(COMM, NEW_MESH_H5, "r")
hdf_tgt_in.read(tgt_mesh, "/mesh", False)

nsd_tgt = tgt_mesh.topology().dim()
tgt_dom = MeshFunction("size_t", tgt_mesh, nsd_tgt, 0)
hdf_tgt_in.read(tgt_dom, "/domains")  # require domains as requested
hdf_tgt_in.close()

V_tgt = VectorFunctionSpace(tgt_mesh, "CG", 1)  # displacement space on target mesh

# ────────────────────── OPEN INPUT/OUTPUT FILES ────────────
snap_h5 = HDF5File(COMM, SNAPSHOTS_H5, "r")

h5_out = HDF5File(COMM, OUT_H5, "w")
h5_out.write(tgt_mesh, "/mesh")
h5_out.write(tgt_dom,  "/domains")

xmf_disp = XDMFFile(COMM, OUT_XDMF_DISP)
xmf_disp.parameters["flush_output"] = True
xmf_disp.parameters["functions_share_mesh"] = True
xmf_disp.parameters["rewrite_function_mesh"] = False
xmf_disp.write(tgt_mesh)  # write mesh once

# Buffers
u_src = Function(V_src); u_src.rename("Displacement", "Displacement")
u_src.set_allow_extrapolation(True)
u_tgt = Function(V_tgt); u_tgt.rename("Displacement", "Displacement")


if RANK == 0:
    print(f"[info] Regridding {NUM_SNAPSHOTS} displacement snapshots onto NEW mesh...")

# ────────────────────── LOOP: read → interpolate → write ───
t0 = 0.0
for i in range(NUM_SNAPSHOTS):
    t_snap = t0 + i * DT_SNAP
    if RANK == 0:
        print(f"[{i+1}/{NUM_SNAPSHOTS}] t = {t_snap:.6f} s")

    # read displacement on source mesh
    snap_h5.read(u_src, f"/displacement/{i}")

    # interpolate to target mesh
    LagrangeInterpolator.interpolate(u_tgt, u_src)

    # write to consolidated HDF5
    h5_out.write(u_tgt, f"/displacement/{i}")

    # write to XDMF series for ParaView
    xmf_disp.write(u_tgt, t_snap)

# ────────────────────── CLOSE ──────────────────────────────
snap_h5.close()
xmf_disp.close()
h5_out.close()

if RANK == 0:
    print("[done] Wrote:")
    print("  •", OUT_H5,        " (mesh, domains, displacement/{i})")
    print("  •", OUT_XDMF_DISP, " (ParaView time series)")

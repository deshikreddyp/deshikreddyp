Reference Values

You often see a tab named Reference Values in software packages such as Abaqus and Ansys. These are characteristic quantities used for normalization, coefficient reporting, and post-processing.

Problems are frequently non-dimensionalized for analysis and comparison. Typical reference data include free-stream velocity, characteristic length, area, density, pressure, temperature, viscosity, and specific heats.

Ansys

In Ansys Fluent, reference values can be entered manually or computed from a selected boundary zone. Common reference quantities are area, density, enthalpy, length, pressure, temperature, velocity, dynamic viscosity, and ratio of specific heats.

For 2D simulations, Depth is an additional quantity used when reporting forces and flux-related outputs (e.g., flow rate, mass flow rate, and integrated quantities).

In the Set Units dialog box, depth units are specified separately from length units.

If you use Compute From, only quantities implied by that boundary condition are populated. Reference length and area are often set manually.

To set values manually, provide each field under the Reference Values panel.

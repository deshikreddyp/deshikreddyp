Reference Values

Normalization and Characteristic Quantities

In simulation software like Abaqus, Ansys, and OpenFOAM, "Reference Values" are characteristic quantities used for normalization, reporting nondimensional coefficients (like $C_d$ or $C_l$), and post-processing.

💡 Nondimensionalization: Problems are frequently scaled to characteristic values to allow for comparison across different scales. Typical reference data include free-stream velocity ($U_\infty$), characteristic length ($L$), area ($A$), and density ($\rho$).

Industry Standard: Ansys Fluent

In Fluent, reference values computed from a boundary zone or entered manually include:

  • Density and Velocity: Typically taken from the inlet or far-field.
  • Area and Length: Used for computing force coefficients ($C_d = \frac{F_d}{\frac{1}{2}\rho U^2 A}$).
  • Dynamic Viscosity: Used for Reynolds number calculation.

⚠️ 2D Simulations: For 2D cases, the Depth (spanwise thickness) must be specified correctly to report accurate force magnitudes. Default depth is usually $1.0$.

If using the "Compute From" option, Fluent only populates values derived from that boundary condition. Geometric values like Reference Length and Area almost always require manual entry.

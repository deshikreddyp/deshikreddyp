STL to BRep (stl2brep.py)

Category: Meshing · Geometry Conversion · Python · GMSH/OCC

In CFD mesh generation, raw STL files often lack the topological information required for advanced boolean operations or conformal meshing. Converting these surface triangulations into a BRep (Boundary Representation) allows GMSH's OpenCascade kernel to treat each facet as a geometric entity.

Functional Overview.

This script iterates through every triangle in an STL file, creates geometrical points and lines, and then defines plane surfaces to construct a closed volume. This is essential for converting "dead" STL meshes into "live" CAD geometries.

import gmsh
import sys
import numpy as np

# Usage: python stl2brep.py model.stl
gmsh.initialize()
gmsh.open(sys.argv[1])

# Retrieve mesh data
typ = 2 # 3-node triangles
elementTags, elementNodes = gmsh.model.mesh.getElementsByType(typ)
nodeTags, nodeCoord, _ = gmsh.model.mesh.getNodesByElementType(typ)
edgeNodes = gmsh.model.mesh.getElementEdgeNodes(typ)

# Create BRep model
gmsh.model.add('Geometry')
nodes = dict(zip(nodeTags, np.reshape(nodeCoord, (len(nodeTags), 3))))
for n_tag, coord in nodes.items():
    gmsh.model.occ.addPoint(coord[0], coord[1], coord[2], tag=n_tag)

# Construct surfaces
element_edges = np.reshape(edgeNodes, (len(elementTags), 3, 2))
allcurves = {}
allsurfaces = []

for edges in element_edges:
    curves = []
    for edge in edges:
        ed = tuple(np.sort(edge))
        if ed not in allcurves:
            tag = gmsh.model.occ.addLine(edge[0], edge[1])
            allcurves[ed] = tag
        curves.append(allcurves[ed])
    cl = gmsh.model.occ.addCurveLoop(curves)
    allsurfaces.append(gmsh.model.occ.addPlaneSurface([cl]))

# Finalize volume
sl = gmsh.model.occ.addSurfaceLoop(allsurfaces)
gmsh.model.occ.addVolume([sl])
gmsh.model.occ.synchronize()
gmsh.write("output.brep")
gmsh.finalize()
Performance Constraint.

Because each STL facet becomes a NURBS surface, this approach is extremely memory-intensive. For geometries with more than 10,000 facets, consider using a high-memory cluster or GMSH's ClassifySurfaces approach instead.

Installing GMSH on cluster

Making conformal meshes, especially of complex geometries might demand excessive resources. Generating a conformal mesh on 6000 faceted geometry takes approximately 4 hrs on a 32 GB RAM intel processor. Biological simulations often requires the geometry to be very accurate, which means more faceted goemetry, and more resources.

Although GMSH features are still in experimental stage in parallel, one can use the cluster resources in serial.

Required modules for GMSH working,

  1. RCAC
  1. mesa-glu/9.0.2 (available only on Negishi)
  1. mpc
  1. mpfr (optional)

Install GMSH using the pip+

pip install gmsh

Run GMSH

gmsh
